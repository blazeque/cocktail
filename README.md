# Test-Bar

System to adminstrate drink order

### Installation

#### Start MongoDB Locally

```
$ sudo mongod
```

#### Run Backend

```
$ cd Backend/
$ yarn install
$ yarn run dev
```

#### Run Menu Page

```
$ cd Menu/
$ yarn install
$ yarn run dev
```

#### Run Kitchen System

```
$ cd Kitchen/
$ yarn install
$ yarn run dev
```

### Usage

To acess Menu Page
http://localhost:3000/

To acess Kitchen System
http://localhost:8080/
